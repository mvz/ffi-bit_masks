## Description

FFI plugin which adds support for bit-wise fields (or flags) to FFI.

## Examples

    require 'ffi/bit_fields'

## Requirements

* [ffi](https://github.com/ffi/ffi#readme) ~> 1.0

## Install

    $ gem install ffi-bit_fields
