## Description

FFI plugin which adds support for bitmasked types (or flags) to FFI.

## Examples

    require 'ffi/bit_masks'

## Requirements

* [ffi](https://github.com/ffi/ffi#readme) ~> 1.0

## Install

    $ gem install ffi-bit_masks
